HTML

Introduction to HTML and its importance in web development

Topics Covered

  • Basic tags: html, head, body, headings, paragraphs, lists
  • Tables: Creation of tables using table, tr, td tags
  • Images: Adding images using the img tag
  • Hyperlinks: Linking pages using a tags
  • Forms: Creating forms with input fields, buttons, and dropdowns

Home Task - Create a small project implementing basic HTML concepts

CSS

Styling websites with CSS to enhance visual appeal

Topics Covered

  • Colors: Applying background and text colors
  • Backgrounds: Adding background images and colors
  • Padding and Margin: Understanding spacing around elements
  • Borders and Shadows: Styling elements with borders and shadows

Practical Examples: Hands-on exercises for each topic

Home Task - A small project to apply CSS concepts to a webpage

Bootstrap

Introduction to Bootstrap for responsive web design

Topics Covered

  • Bootstrap Fundamentals: Overview of the framework
  • Grid System: Designing responsive layouts using Bootstrap's grid
  • Forms: Creating responsive forms using Bootstrap classes
  • Using Bootstrap from the Official Website

Home Task - Small project using Bootstrap to design a responsive webpage

JavaScript

Fundamental concepts of JavaScript for interactive web development

Topics Covered

  • Variables: Declaring and initializing variables
  • Conditions: If-else, switch statements
  • Loops: For, while, and do-while loops
  • Functions: Defining and using functions

Home Task - A simple project to reinforce JavaScript concepts

Node.js API Development

Building backend services using Node.js and Express

Topics Covered

  • Introduction to Node.js and setting up a basic Node server
  • Creating APIs: GET, POST, PUT, DELETE, and PATCH methods
  • API Testing: Using Postman for testing
  • Storing Data: Connecting MongoDB for storing and retrieving data
  • Authentication and Authorization: Securing APIs
  • File Upload: Implementing document upload functionality using APIs

Home Task - Develop a simple API as a project

React.js

Developing dynamic front-end applications using React

Topics Covered

  • React Fundamentals: Introduction to components, JSX, and state
  • Component Development: Building reusable components
  • Hooks: Understanding useState, useEffect, and other hooks
  • Consuming APIs: Integrating APIs with React front-end
  • Full-stack Integration: Connecting React with Node.js APIs

Final Project - Complete a full-fledged project to earn certification
